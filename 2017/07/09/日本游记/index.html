日本游记

发表于 2017-07-09

题记：世界那么大，只有出去了才知道有多精彩

本来打算毕业旅行就去日本的，然而忽然发现根本没有时间，还好公司组织了一波 outing。

印象

从前就知道日本是一个高度自律的国家。但是只有自己亲眼见到了，才会被很多细节深深的震撼。

  1. 在上自动扶梯的时候，所有人都会自动站在左侧（日本的默认方向和国内相反，车都是靠左行驶的），留右边提供给需要快速上楼的人。就算人多的时候也会默认在左侧排队。而不会堵住右侧。
  2. 所有不能停车，需要即走即停的地方，都需要有规定的时间，基本上5分钟之内就要乘完车，然后出发。基本不会存在超时的情况。从而影响别人的行程。（日本遵从着不麻烦别人的这一个准则，进行生活）
  3. 在封闭空间的公共场合都不会高谈阔论，从而不影响别人乘车的心情
  4. 在开车的时候基本上不会按喇叭，开车的司机都会“慢慢”的开车，很少会出现急刹车等各种情况（不要问我头文字D，老司机表示不知道弯道超车是什么）
  5. 日本马路上很少存在灰尘，同时会发现所有车子（包括卡车）都炒鸡干净

除了高度的自律，日本身边产品对于细节的打磨更加令人惊叹：

  1. 日本马桶圈，嗯，就是那个加热屁屁，洗屁屁的东西，酒店里面都有！
  2. 酒店里面放置沐浴液的架子和人头的固定架子都放了2层，方便够不到最上层的架子的小朋友也能正常使用。
  3. 所有日产的包装都存在明显的口子，方便用户撕开包装，就算没有口子，外部的袋子也会方便撕开。
  4. 大多数景点附近都会存在救援按钮或者心脏起搏器，方便有人身体不适的时候，能够及时报警，方便进行救助。

思考

在日本的几天虽然很短，但是让我印象深刻。但是仔细想来，感觉日本和国内现在的发展，或者说杭州的发展有着很大的区别。日本是所有人都按照自己的本分去做自己该做的事情，拿自己该拿的钱。（毕竟就算是流浪汉，什么都不干，只要不给社会惹麻烦，政府每个月都能给大概3k的工资到手上，基本上吃饱是没啥问题的）而对于其他行业的工人，只要你愿意吃苦耐劳的干活，就算你是司机，一个月基本上也能拿到上万的工资（转化成人民币后），但是总感觉缺少一点活力。最让人能感受到活力的，可能就是商场或者公共场所了吧。有时候比如说上野公园，我走在路上的时候，真的是安静的可怕。
仔细想想，可能就是因为互联网的缘故吧，因为杭州的互联网发展的迅速，导致所有的行为，需要进行交流的东西都可以通过互联网进行相互联系，包括人们日常的衣食住行，就算是走进小弄堂里面的小店都可以支持支付宝付款（这条五毛）。人们的生活十分方便。
再回过头去看看日本的互联网，简单的进行举个例子进行对比：日本的动漫视频网站 ニコニコ 是一开始国内两大动漫网站 ACFun 和 Bilibili 争相模仿的，但是现在再回过头去看看他们的界面。



高下立判。总感觉日本在接下去的互联网革命（人工智能相关）的过程中会出现严重的落后。（不过希望到时候，日本的周边能够便宜一点，否则连塑料小人都买不起，人生还有什么意义！！！2333333333）
